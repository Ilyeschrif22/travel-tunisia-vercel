# Travel Tunisia

A React-based web application for discovering authentic tours and cultural experiences in Tunisia.

## About

Travel Tunisia is a tourism platform that showcases the beauty of Tunisia through expert tours featuring ancient ruins, magnificent beaches, and vibrant culture. The platform connects travelers with local guides for authentic Tunisian experiences.

**Live Demo:** [https://traveltunisia.vercel.app/](https://traveltunisia.vercel.app/)

## Features

- **Destination Showcase**: Explore Tunisia's most visited locations including beaches, historic sites, and natural wonders
- **Local Guides**: Connect with friendly local guides who share authentic insights into Tunisian culture
- **Affordable Tours**: Budget-friendly tour packages designed for every traveler
- **Flexible Itineraries**: Personalized tours that adapt to your interests and pace
- **Cultural Immersion**: Rich experiences highlighting Tunisia's history, traditions, and natural beauty

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm start`
4. Open [http://localhost:3000](http://localhost:3000) in your browser
